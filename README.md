
# Cheese Boards CR

Sitio estático accesible, multilenguaje (ES/EN/PT) y responsive.

## Estructura
```
cheese_boards_cr/
├─ index.html
├─ styles.css
├─ script.js
├─ i18n.js
└─ assets/
   ├─ logo.png
   └─ hero.jpg
```

## Publicación rápida
- **GitHub Pages**: repo → Settings → Pages → Deploy from branch.
- **Netlify**: arrastra la carpeta al panel.
- **Vercel**: importa la carpeta y despliega.

## Personaliza
- Cambia colores en `styles.css` (variables `:root`).
- Reemplaza `assets/logo.png` y `assets/hero.jpg` por tus imágenes.
- Edita textos en `i18n.js`.
